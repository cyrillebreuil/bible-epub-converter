(c) « Seigneur de l’univers », première occurrence de l’expression dans la Bible. En hébr. : YHWH çevaoth, « Seigneur des armées », expression rendue de diverses manières par la Bible grecque. À l’origine, ces armées sont, sans doute, les armées d’Israël, conduites par l’Arche (1 S 4,4 ; 15,2 ; 17,45 ; 2 S 5,10 ; 6,2.18) ; puis les armées célestes : puissances cosmiques, astres (Gn 2,1 ; Jg 5,20 ; Is 34,4 ; Jr 33,22 ; etc.), ce qu’aujourd’hui nous appelons l’univers ; de là, la traduction : « Seigneur de l’univers » (1 S 1,11 ; 2 S 7,8.26-27), qui est la plus fréquente dans les autres livres.
