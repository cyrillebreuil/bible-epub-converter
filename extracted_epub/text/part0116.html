Notes

(a) « Rendez hommage au fils par un baiser », om. ; hébr. : texte et sens incertains ; Versions : « Recevez l’instruction ».

(b) Pour indiquer les selah, terme de sens incertain souvent traduit par « pause », on a employé le signe « P ».

(c) « son », hébr. : « mon ».

(d) « Jusqu’aux cieux [...] adversaire », texte obscur ; le texte massorétique et les Versions coupent à la fin du v. 2.

(e) Sens incertain ; on peut aussi comprendre : « sur [l’air de] mourir pour le fils ».

(f) « que tremble le mortel », on peut aussi comprendre : « que le mortel n’opprime plus ».

(g) Le grec et la Vulgate ajoutent : « et je célébrerai le nom du Seigneur, le Très-Haut ».

(h) « mon cœur », litt. : « mes reins ».

(i) « corruption », d’après les Versions (cf. Ac 13,35).

(j) « De tous côtés », litt. : « Il les dispersa », à savoir : les ennemis.

(k) « des mers », litt. : « des eaux ».

(l) « ma nuit », litt. : « ma ténèbre ».

(m) « de bronze », om.

(n) « Par ton bouclier tu m’assures la victoire », litt. : « Tu me donnes ton bouclier de salut ».

(o) « des années sans fin », litt. : « à jamais et toujours ».

(p) « Il comptait », Versions.

(q) « Tu me mènes », d’après les Versions.

(r) « Ils me percent », d’après les Versions ; hébr. obscur.

(s) « Tu seras ma louange », litt. : « De toi [vient] ma louange ».

(t) « de l’ombre », om.

(u) « le monde […] la terre », litt. : « la terre […] le monde ».

(v) « idoles », litt. : « vanité », ou « néant ».

(w) Certains corrigent le texte hébr. du v. 6b et traduisent avec le grec : « de ceux qui cherchent la face du Dieu de Jacob ».

(x) Le v. 7c « en raison de ta bonté, Seigneur » a été placé à la fin du v. 5 pour assurer la régularité de la strophe.

(y) « mon rempart », litt. : « mon bouclier ».

(z) « ma chair a refleuri », avec les Versions.

(aa) « tous », d’après les Versions (« quiconque »).

(ab) « viens », litt. : « hâte-toi ».

(ac) « Mes jours », litt. : « Mes temps ».

(ad) « le peuple […] la nation », litt. : « la nation […] le peuple ».

(ae) « De tout mon être, je dirai », litt. : « Tous mes os diront ».

(af) « Les doux », avec les Versions (cf. Mt 5,5).

(ag) « tu as ouvert mes oreilles », les principaux mss grecs ont : « tu m’as formé un corps » (cf. He 10,5).

(ah) « est divin », on peut aussi traduire : « ô Dieu ».

(ai) « la préférée », d’autres traduisent : « la reine » (cf. Ne 2,6).

(aj) « toute », om.

(ak) « ils feront place au juste », litt. : « les justes domineront sur eux ».

(al) « je te prends à témoin », on peut aussi comprendre : « je témoigne contre toi ».

(am) « sur le chemin qu’il aura pris », certaines versions, suivies par la Néo-Vulgate, ont : « à celui dont la voie est intègre ».

(an) « pécheur dès le sein » : l’intention du psalmiste n’est pas de renvoyer la faute sur le passé de sa mère. Le texte hébreu renvoie peut-être au passé d’Israël, comme Ez 16.

(ao) « malgré », litt. : « à cause de ».

(ap) « prêt », litt. : « affermi ».

(aq) « Je n’ai mon repos », d’après plusieurs mss qui donnent l’indicatif au lieu de l’impératif ; texte reçu : « mon âme se repose ».

(ar) « dans Sion » : c’est ainsi que les Versions ont compris le texte hébreu.

(as) La coupure du vers a été déplacée ; « poids », litt. : « affaire » ou « cas ».

(at) « l’air de », add.

(au) « m’a perdu », litt. : « m’a mangé ».

(av) « En ces jours-là », litt. : « En ses jours ».

(aw) « aux portes de Sion », add. avec les Versions.

(ax) « Détruisons », d’après le syriaque ; hébr. : « leurs descendants ont dit ».

(ay) « Sur l’air de », add.

(az) « figuiers », on traduit habituellement : « sycomores ».

(ba) « rejeton », avec le syriaque ; hébr. : « le fils » (cf. v. 18 b).

(bb) « Ashour » : il ne s’agit pas de l’Assyrie (Assour), mais d’une tribu de Palestine.

(bc) « Dieu », add.

(bd) « Dieu », add.

(be) « dans cette nuit où je crie en ta présence », litt. : « le jour, j’ai crié dans la nuit devant toi ».

(bf) « Leur plus grand nombre », Versions.

(bg) « Qui peut t’adorer dans tes fureurs ? », litt. : « Comme ta crainte, ton courroux ».

(bh) « Dieu qui fais justice », litt. : « Dieu des vengeances ».

(bi) « Dieu qui fais justice », litt. : « Dieu des vengeances ».

(bj) D’autres traduisent ainsi le v. 10 : « Celui qui corrige les peuples n’avertirait-il pas, lui qui donne aux hommes la connaissance ? »

(bk) « Toi, tu es le même », allusion au nom divin.

(bl) « marmottes », litt. : « damans » (cf. Lv 11,5 ; Dt 14,7 ; Pr 30,26).

(bm) « royaume », litt. : « peuple ».

(bn) « son action est tenue », litt. : « cela lui fut compté ».

(bo) « prêt », litt. : « affermi ».

(bp) « présente », comme pour un geste d’investiture. Pour le même verbe, les Versions donnent « étend ».

(bq) « le sceptre de ta force », Versions.

(br) « Comme la rosée [...] je t’ai engendré », texte difficile ; hébr. : « Depuis le sein, depuis l’aurore, à toi la rosée de la jeunesse ( ?) ». Versions : « Du sein, avant l’astre de l’aurore, je t’ai engendré. » Nous suivons la Néo-Vulgate (1re éd.) : « Ex utero matutini velut rorem genui te. »

(bs) « roi », add.

(bt) « les vivants », add. avec le grec.

(bu) « et » ; Versions : « c’est pourquoi ».

(bv) « s’est montré le plus fort », selon l’interprétation du texte hébreu par la Septante.

(bw) « chant », autre sens possible : « protection ».

(bx) « promesses », (plus de quinze fois dans ce psaume) ; Versions : logion, eloquium (singulier ou pluriel).

(by) « exil », équivalence pour « Mèshek » ; « désert », équivalence pour « Qédar » (noms de lieux).

(bz) « Israël doit », litt. : « règle pour Israël ».

(ca) Les termes « bonheur » et « paix » ont été inversés dans ce verset.

(cb) « est le nom », litt. : « est dans le nom ».

(cc) « idole », litt. : « perversion ».

(cd) « soumission ». Versions : « douceur ».

(ce) « tu as entendu les paroles de ma bouche », add. avec les Versions.

(cf) « poisons ». Versions : « huile de l’impie », mais le texte hébreu joue peut-être sur l’homographie ; hébr. : de « tête » et « poison ».

(cg) « tous », om.

(ch) « à l’heure du triomphe », litt. : « sur vos couches » ; le texte hébreu contient peut-être une allusion à l’épreuve.

(ci) Les stiques de ce verset ont été intervertis.
