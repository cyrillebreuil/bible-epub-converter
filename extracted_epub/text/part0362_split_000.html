Entrée en catéchuménat :

PS : Ps 50,3… ; Ps 24,4… ; Ps 90,1…

ÉV : Mt 4,1-11 ; Mc 1,12-15 ; Lc 4,1-13.

Scrutins

Premier scrutin, on prend les lectures du troisième dimanche du Carême de l’année A avec leurs chants.

Thème de l’eau vive : Ex 17,3-7 ; Ps 94,1… ; Rm 5,1-2.5-8 ; ÉV : Jn 4,5-42.

Troisième scrutin, on prend les lectures du cinquième dimanche du Carême de l’année A avec leurs chants.

Thème de la vie : Ez 37,12-14 ; Ps 129,1… ; Rm 8,8-11 ; ÉV : Jn 11,1-45.

Reddition du Symbole : Mt 16,13-17 ; Mc 7,31-37 ; Jn 6,35.63-71.
