IX. DÉDICACE OU BÉNÉDICTION D’UNE ÉGLISE OU D’UN AUTEL

1. Dédicace d’une Église

AT : Ne 8,2-4a.5-6.8-10.

PS : Ps 18B,8…

NT : 1 Co 3,9b-11.16-17 ; Ep 2,19-22 ; He 12,18-19.22-24 ; 1 P 2,4-9 ; au Temps pascal, on peut aussi prendre les lectures suivantes qui sont proposées comme premières lectures lors de l’anniversaire de la dédicace d’une église : Ap 21,1-5a ; Ap 21,9b-14.

ÉV : comme lors de l’anniversaire de la dédicace d’une église.

 

2. Dédicace d’un autel

Premières lectures :

AT (en dehors du Temps pascal) : Gn 28,11-18 ; Jos 8,30-35 ; 1 M 4,52-59.

NT (au Temps pascal) : Ac 2,42-47 ; Ap 8,3-4.

PS : Ps 83,3… ; Ps 94,1… ; Ps 117, 15… ; Ps 118,129… ; Ps 121,1…

Deuxièmes lectures : NT : 1 Co 10,16-21 ; He 13,8-15.

ÉV : Mt 5,23-24 ; Jn 4,19-24 ; Jn 12,31-36a.

 

3. Bénédiction d’un calice et d’une patène

NT : 1 Co 10,14-22a ; 1 Co 11,23-26.

PS : Ps 15,5… ; Ps 22,1…

ÉV : Mt 20,20-28 ; Mc 14,12-16.22-26.

 

4. (Propre au Lectionnaire français) Pose de la première pierre d’une Église

AT : 1 R 5,16-32 ; Is 28,16-17.

NT : Ac 4,8-12 ; 1 Co 10,1-6.

PS : Ps 23,1… ; Ps 41,3… ; Ps 86,1… ; Ps 99,1… ; Ps 117,1…

ÉV : Mt 7,21-29 ; Mt 16,13-18 ; Mc 12,1-12 ; Lc 6,46-49
